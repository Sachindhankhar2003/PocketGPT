# 🚀 PocketGPT

**PocketGPT — Smart answers anywhere**

PocketGPT is a modern AI chatbot web application that provides fast and intelligent responses using powerful AI models.  
It allows users to chat with AI, analyze documents, and manage conversations through a clean dashboard interface.

---

# ✨ Features

- 🤖 AI Chat Assistant
- 💬 Conversation History
- 📄 Document Upload & Analysis
- 📊 Dashboard Analytics
- 📱 Mobile Responsive Design
- ⚡ Fast AI Responses
- 🗂 Chat Management (Delete / View)
- 🎨 Modern Dark UI

---

# 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI Integration
- OpenRouter API
- Llama / Mistral / Gemma models
